# Flutter

Flutter and Dart practice — a few small learning projects, not finished products.

## Projects

- `flutter_app` — a bottom-navigation app skeleton (Home/User pages, a dark-mode toggle,
  shared `ValueNotifier` state) plus the course steps that built it.
- `ninja_app` — a first single-screen app for practicing layout widgets.
- `dart-tuto` — plain Dart language basics.

Run a Flutter app with `flutter run` (after `flutter pub get`) from its folder; run the Dart
scripts with `dart run <file>.dart`.
